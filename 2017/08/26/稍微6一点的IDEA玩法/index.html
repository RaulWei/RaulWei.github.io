稍微6一点的IDEA玩法

发表于 2017-08-26 | 分类于 Technique |

一开始进组对什么都不熟悉，乖乖地按照新人手册搭环境，跑项目，到如今也有一个多月了，别别扭扭地用了这么久eclipse，终于在这周换了IDEA。两个字，真爽！太他妈智能了！代码自己蹭蹭地往外冒，感觉都不用自己写了。其实以前也用过一段时间IDEA，但是学生嘛，写点玩具项目，也没什么需求接触IDEA的各种功能。那天看到凯哥一顿行云流水的操作，很眼馋，于是决定挖掘和记录一下IDEA在工作中比较常用的功能快捷键，希望可以帮助提升工作效率和体验。

设置相关

快捷键 功能
Alt + Ctrl + s 打开Settings
Alt + Ctrl + Shift + s 打开Project Structure，也可以先选中Project，再按F4，效果一样

搜索相关

快捷键 功能
Ctrl + e 找到最近访问的文件
Ctrl + Shift + e 找到最近编辑的文件
Ctrl + Click 按住Ctrl，再点击它就会跳转到对应的源码中去
敲两下Shift 弹出搜索框，根据类名寻找类，或根据资源名寻找资源，或根据/文件夹名寻找文件夹
Ctrl + Shift + A 调出Enter action or option name，帮助搜索行为，如搜REST Client可调用它
Ctrl + F12 悬浮显示该类的Structure，如果记得方法名，想跳转过去，可直接在焦点位于悬浮窗时输入，匹配后回车，贼方便

编辑相关

快捷键 功能
Shift + Enter 在行中任意位置执行此快捷键，可另起一行，并跳转过去
Ctrl + Shift + Enter 在行中任意位置执行此快捷键，可快速补全行尾分号
Shift + F6 重命名类、方法、变量
Ctrl + Shift + z 取消撤销
Alt + Insert 可以选择自动插入构造器、getter、setter等
Alt + Enter 可以选择提示的建议来优化代码
